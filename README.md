# 在庫・売上管理マネージャー

モバイルファーストの在庫・売上管理アプリケーション（PWA）

## 機能

- ✅ 在庫管理（商品の追加・編集・削除）
- ✅ 入出庫管理（入庫・出庫記録、履歴追跡）
- ✅ 売上管理（売上登録、売上履歴、キャンセル・返品）
- ✅ ダッシュボード（統計サマリー、在庫不足アラート）
- ✅ 分析・レポート（売上推移、商品別ランキング、グラフ）
- 🔄 PWA対応（オフライン動作、ホーム画面に追加可能）
- 🔄 Excel/CSV インポート/エクスポート（開発中）

## 技術スタック

- **React 18** - UIフレームワーク
- **TypeScript** - 型安全な開発
- **Vite** - 高速なビルドツール
- **TailwindCSS** - スタイリング
- **Dexie.js** - IndexedDBラッパー（オフラインファースト）
- **TanStack Query** - サーバー状態管理
- **Recharts** - グラフ表示
- **vite-plugin-pwa** - PWA機能

## セットアップ

```bash
# インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## PWAアイコンの作成

PWAアイコンを生成するには、以下の手順を実行してください：

1. `public/icon.svg` をお気に入りのデザインツールで編集するか、オンラインツールでPNGに変換
2. 以下のサイズのPNGアイコンを作成して `public/` ディレクトリに配置：
   - `icon-192x192.png`
   - `icon-512x512.png`

オンライン変換ツールの例：
- https://cloudconvert.com/svg-to-png
- https://convertio.co/ja/svg-png/

## 使用方法

1. **ダッシュボード**: 統計情報、在庫不足アラート、クイックアクション
2. **在庫管理**: 商品の一覧・追加・編集・削除
3. **売上管理**: 売上登録、売上履歴、キャンセル・返品
4. **分析**: 売上推移、商品別ランキング、支払い方法別比率

## オフライン動作

このアプリはPWAとして設計されているため、オフラインでも動作します。
- すべてのデータはローカルのIndexedDBに保存されます
- オフラインでも在庫管理・売上登録が可能です
